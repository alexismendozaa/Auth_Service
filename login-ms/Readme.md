# 🛡️ LOGIN Microservice

This microservice handles user authentication.

## 🚀 Getting Started

### Requirements

- [Node.js](https://nodejs.org/) (v16+ recommended)

---

## ▶️ How to Run

```bash
cd login-ms
npm install
npm start

🔄 API Endpoints
POST /api/login – Authenticates the user and returns a JWT token.

POST /api/logout – Logs the user out.
