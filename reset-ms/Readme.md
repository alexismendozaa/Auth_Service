# 🔑 RESET Microservice

This microservice allows users to reset their passwords.

## 🚀 Getting Started

### Requirements

- [Node.js](https://nodejs.org/) (v16+ recommended)

---

## ▶️ How to Run

```bash
cd reset-ms
npm install
npm start

🔄 API Endpoints
POST /api/reset – Resets the user's password.
